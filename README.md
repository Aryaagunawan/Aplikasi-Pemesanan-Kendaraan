# ArmadaPro – Enterprise Vehicle Management System

ArmadaPro adalah aplikasi **Enterprise Vehicle Management System** berbasis **Fullstack JavaScript** yang digunakan untuk mengelola armada kendaraan operasional perusahaan.

---

## ✨ Features
- 🚘 Vehicle Management
- 📝 Booking System
- ✅ Multi-Level Approval
- 📊 Dashboard Monitoring
- 📄 Reports & Logs

---

## 🧱 Tech Stack
**Frontend**
- React.js (Vite)
- Tailwind CSS
- Axios

**Backend**
- Node.js
- Express.js
- MySQL
